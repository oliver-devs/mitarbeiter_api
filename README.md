# Mitarbeiter API & Management System

Ein Fullstack-Projekt zur Verwaltung von Mitarbeitern und Abteilungen, inklusive Dashboard und sicherer Authentifizierung.

## 🚀 Features

* **Dashboard:** Zentrale Übersicht über die Unternehmensstruktur.
* **Mitarbeiter-Verwaltung:** Vollständiges CRUD (Erstellen, Lesen, Aktualisieren, Löschen) von Mitarbeiterprofilen.
* **Abteilungs-Management:** Organisation von Mitarbeitern in verschiedenen Abteilungen.
* **Sicherheit:** JWT-basierte Authentifizierung mit Login/Logout und geschützten Routen.
* **Responsive UI:** Modernes Design mit Angular Material und Sidebar-Navigation.

## 🛠 Technologien

### Backend
* **Python / Django** (Web Framework)
* **Django REST Framework** (API Schnittstelle)
* **SQLite** (Datenbank)

### Frontend
* **Angular** (Framework)
* **Angular Material** (UI Komponenten)
* **RxJS** (Asynchrone Datenströme)
